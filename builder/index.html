۲. راهنمای استفاده 2. How to Use

  1. تمپلیت را از گیت‌هاب دانلود کنید
  2. Download the template from GitHub
  3. فایل config.js دانلود شده را جایگزین assets/js/config.js در تمپلیت کنید
  4. Replace assets/js/config.js in the template with your downloaded file
  5. عکس پروفایل خود را در assets/img/profile.jpg قرار دهید
  6. Put your photo in assets/img/profile.jpg
  7. عکس گواهینامه‌ها را در assets/img/certs/ قرار دهید
  8. Put certificate images in assets/img/certs/
  9. index.html را در مرورگر باز کنید → پورتفولیوی شما آماده است!
  10. Open index.html in a browser → your portfolio is ready!

۳. انتشار در گیت‌هاب پیج 3. Deploy to GitHub Pages

# Create a repo named username.github.io git init git add -A git commit -m "my portfolio" git branch -M main git remote add origin https://github.com/username/username.github.io.git git push -u origin main # → Your site is live at https://username.github.io
